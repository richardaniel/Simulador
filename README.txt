Para ejecutar el comando make en la terminal ocupamos instalar chocolately (gestor de paquetes para WIndows) ,se puede obtener en :
https://community.chocolatey.org/ 

Posteriro a esto ejecutamos 
$ choco install make 

Y listo podemos utilizar el comando make
